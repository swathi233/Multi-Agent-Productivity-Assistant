# 🤖 AI Personal Assistant (Multi-Agent System)

## 🚀 Overview
This project is an AI-powered personal assistant that helps users manage tasks, notes, meetings, and reminders in a single platform.

It integrates with Google services such as Gmail, Google Calendar, and Google Tasks to provide real-world functionality.

---

## 🎯 Features
- 📌 Task Management (Google Tasks sync)
- 📝 Notes (saved to Gmail)
- 📅 Meeting Scheduler (Google Calendar)
- 🎂 Birthday reminders
- 🎤 Voice input for tasks
- 🔐 User authentication

---

## 🧠 Tech Stack
- Python
- Streamlit
- SQLAlchemy (SQLite DB)
- Google APIs (Gmail, Calendar, Tasks)
- SpeechRecognition
